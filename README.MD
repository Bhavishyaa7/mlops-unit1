# MLOps Unit 1 Project

## Project Objective
Build a simple ML model using Logistic Regression and demonstrate MLOps workflow.

## Dataset Used
Titanic dataset from:
https://raw.githubusercontent.com/datasciencedojo/datasets/master/titanic.csv

## Steps to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the project:
   python src/experiment2.py
